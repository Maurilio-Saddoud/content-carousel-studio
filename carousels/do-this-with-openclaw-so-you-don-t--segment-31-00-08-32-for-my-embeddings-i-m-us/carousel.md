eyebrow: MODEL STRATEGY
# Serious agent stacks are multi-model by default.

One model for everything is convenience,
not architecture.

---

eyebrow: WHERE QUALITY LIVES
# Use the best model where judgment matters.

Planning.
Hard coding.
Security review.
High-stakes decisions.

---

eyebrow: WHERE EFFICIENCY LIVES
# Use cheaper models where repetition matters.

Search.
Background tasks.
Embeddings.
Routine classification.

---

eyebrow: THE OPERATING RULE
# The goal is not “cheapest.”

It is:
best model for the decision,
cheapest model for the repetition.

That is how you keep both quality and budget.

---

eyebrow: THE REAL TAKEAWAY
# Expensive and mediocre usually means bad routing.

It is probably not a model problem.

It is a routing problem.
